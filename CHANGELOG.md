# Changelog

All notable changes to this project will be documented in this file.

## [1.3.0] - 2025-11-25

### Added
- **Floating Icon**: Automatically show a floating icon when text is selected to trigger translation.

## [1.2.0] - 2025-11-24

### Added
- **OpenRouter Integration**: Support for accessing any AI model via OpenRouter API.
- **Dynamic Model Selection**: Dropdown menu in settings to fetch and select available models from OpenRouter.
- **Multi-language Voice Support**: Text-to-Speech now automatically detects the language of the text and uses the appropriate voice.
- **Localized Explanations**: AI explanations are now provided in the user's selected target language.

### Changed
- Updated settings UI to support model selection and OpenRouter API key.
- Improved background script to handle dynamic API endpoints.
